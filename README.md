# Project: Recursion

## Introduction

This project is a practical exercise in recursion, developed as part of the JavaScript Course on The Odin Project. It covers two classic computer science problems — the Fibonacci sequence and Merge Sort — each implemented using both iterative and recursive approaches where applicable.

## Initial Project Goals

- Implement the Fibonacci sequence using an iterative approach.
- Implement the Fibonacci sequence using a recursive approach.
- Implement the Merge Sort algorithm using a recursive divide-and-conquer approach.
- Write unit tests for all implemented functions using Jest.
- Strengthen understanding of recursion, base cases, recursive steps, and the Recursive Leap of Faith.

## Technologies Used

- JavaScript (ES6+)
- Jest (unit testing)
- Babel (@babel/preset-env)
- Git & GitHub

## Skills Demonstrated

- **Recursion**: Identifying base cases and recursive steps to solve problems.
- **Divide and Conquer**: Breaking down the merge sort problem into smaller subproblems.
- **Recursive Leap of Faith**: Trusting that smaller recursive calls return correct results and focusing only on the current step.
- **Test-Driven Development (TDD)**: Writing tests before implementing functions.
- **Array manipulation**: Using `slice`, `concat`, `push`, and `shift` to manipulate arrays.

## Lessons Learned

- **Biggest challenge**: Understanding how to implement merge sort recursively, specifically the separation of concerns between `mergeSort` (splitting) and `merge` (combining two sorted arrays).
- **How I overcame it**: Applying the 5-step framework for solving recursive problems — finding the simplest input, visualizing examples, relating hard cases to simpler ones, generalizing the pattern, and writing code.
- **General lesson**: The Recursive Leap of Faith is a powerful mindset — once you find the general pattern, trust that smaller recursive calls work and focus only on the current step.
